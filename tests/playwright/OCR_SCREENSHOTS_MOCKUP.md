## How to Generate Real Screenshots

Since Playwright cannot run in this CI environment, follow these steps to generate actual screenshots:

### Option 1: Run Locally

1. Start the application:
   ```bash
   cd /home/runner/work/Privatekonomi/Privatekonomi/src/Privatekonomi.Web
   dotnet run
   ```

2. Run Playwright tests:
   ```bash
   cd /home/runner/work/Privatekonomi/Privatekonomi/tests/playwright
   npx playwright test ocr-receipt-scanning-screenshots.spec.ts
   ```

3. Screenshots will be in `tests/playwright/screenshots/ocr-receipt-scanning/`

### Option 2: Manual Screenshots

Use browser developer tools to take screenshots manually following the guide in `OCR_SCREENSHOTS_GUIDE.md`.

### Option 3: CI/CD Pipeline

Set up a GitHub Actions workflow with a display server (Xvfb) to run Playwright tests and capture screenshots automatically.
